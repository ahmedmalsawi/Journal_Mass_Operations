How It Works

  1. Open Accounting → Journal Entries.
  2. Select multiple records in the list view.
  3. Click Action and choose the mass operation (Draft/Post/Cancel/Print).
  4. Confirm — the module processes all selected entries.

Easy Installation

  • Install from the Apps menu (no extra configuration).
  • Works with Accounting access rules and standard states.
